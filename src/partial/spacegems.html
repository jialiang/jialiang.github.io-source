Space Gems

Space Gems is a relaxing tile-matching game reminiscence of simpler times when games of this genre wasn't plagued by micro-transactions and in-app advertisements.



▶ Play the game here.



It is made with React, following proper React patterns. However, the React and React-DOM libraries were not used in the end product. Instead, they were replaced by Preact, a "tiny (3kb) implementation of the core value of React". The Preact-Compat compatability library was also added so that the switch would require absolutely no code changes. All this was done to make the end product as light as possible. As a result, the entire source code of the game now weights in at only 27kb gzipped, making it one of the lightest web-based tile-matching game out there. For comparison the React-DOM library alone weights in at a hefty 30kb gzipped.


The beautiful backgrounds featured in the game is courtesy of Reimund Bertrams, who was kind enough to release them under the CC0 license, while the ambient music is produced by Purple Music and licensed under the Creative Commons Attribution License 3.0. Sound effects were provided by Leszek Szary and fromtexttospeech.com.


This project is made locally in Singapore with 100% hand-crafted code.
