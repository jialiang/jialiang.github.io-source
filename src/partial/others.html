Guestbook

Thank you for making it to the end of this website! Before you leave, you can leave an entry in the Guestbook to ackonowledge your visit or to comment on your experience.

Guestbook

The guestbook is hosted in Amazon's Ashburn, Virginia datacenter on an instance managed by Heroku and uses Cloudinary for persistent storage. It is the world's fastest NodeJS guestbook with an in-memory database for sub-millisecond data access and interoperability with native C libraries for super fast request handling.


Load test results using Bombardier (flags: -c 100 -d 5s) ran against a local server to retrieve and list 3 entries are as follows:

  • Average reqs/sec: 11698.23
  • Average latency: 8.54ms

For comparison, a simple ExpressJS Hello World application in the same environment:

  • Average reqs/sec: 9730.06
  • Average latency: 10.26ms

Lucky Cat

Rub on the Lucky Cat to get a lucky number!


The lucky cat acts as a good luck charm for this website. The oval shaped object that it is holding is called a Koban. The characters on the coin read "Sen Man Ryo" which means 10 million Ryo. 1 Ryo was considered equivalent to the amount of rice needed to feed a person for a year. If you would like to know more about Lucky Cats in general, you should give Lucky Maneki Neko a visit.


Instead of some phony pseudo-random number generator, the number generator implemented for the Lucky Cat is a high quality true random number generator based on chaotic tent maps as described by Zhou et al. in this journal article .
